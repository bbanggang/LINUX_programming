### 1. 함수 중복을 설명하시오.

함수 중복 : 하나의 클래스나 범위 내에서 이름은 같짐나 ,매개변수의 개수나 자료형이 다른 여러 개의 함수를 정의하는 것

### 2. 연산자 중복을 설명하시오.

연산자 중복 : 이미 존재하는 연산자를 클래스에 맞게 재정의하는 것

### 3. 연산자 함수란 무엇인가?

연산자 함수 : 연산자 중복 구현을 위한 함수

### 4. 객체 지향형 언어의 특징 중의 하나인 다형성의 의미를 설명하시오.

다형성 : 하나의 함수나 객체가 상황에 따라 여러가지 형태나 동작을 가질 수 있는 능력이나 성질
